# centr-isterika

Статический сайт для караоке-клуба ИСТЕРИКА.

## Локальный запуск

Открой `index.html` напрямую или подними простой сервер, например:

```bash
python -m http.server 4173
```

## Публикация

Для публикации через GitHub Pages:

1. Открой `Settings -> Pages` в репозитории.
2. В `Build and deployment` выбери `Deploy from a branch`.
3. Выбери ветку `main` и папку `/ (root)`.
