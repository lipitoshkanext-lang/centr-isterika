# centr-isterika

Статический сайт для караоке-клуба ИСТЕРИКА.

## Локальный запуск

Открой `index.html` напрямую или подними простой сервер, например:

```bash
python -m http.server 4173
```

После первого пуша публикация будет идти через GitHub Pages workflow в `.github/workflows/deploy-pages.yml`.
